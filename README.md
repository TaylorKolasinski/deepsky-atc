# DeepSky ATC

AI-powered Air Traffic Control simulator using real-world flight data and Unreal Engine visualization.

## Phase 1: Data Acquisition and Simulation Environment Setup

Target Airspace: 150nm x 150nm around KJFK (JFK International Airport)

## Setup
```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

## Project Structure

- `src/` - Core simulation code
- `data/` - Flight data and configuration files
- `docs/` - Technical specifications and documentation
- `tests/` - Validation scripts
